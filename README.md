# 科學視覺化實作集 (Visualization Collection)
本倉庫整合了五項核心電腦圖學實作，範疇涵蓋高階的科學視覺化演算法。透過 **C++** 與 **OpenGL API**，探索如何將複雜的物理數據（如 3D 體積掃描、流體場向量）轉化為直覺且具備互動性的視覺表現。

---

## 目錄

### 1. Iso-Surfaces
* **技術核心**：Marching Cubes 演算法、線性插值、網格提取。
* **開發重點**：將 $256^3$ 的原始純量數據（.raw）轉換為幾何網格。
* **特色功能**：支援不透明骨骼與半透明皮膚的多層渲染，並具備即時平面裁切（Whole/Cross/Slice）功能。

### 2. Volume Rendering Using GPU
* **技術核心**：光線投射法 (Ray Casting)、1D 傳遞函數 (Transfer Function)、3D 紋理映射。
* **開發重點**：無需轉換網格，直接對體素進行採樣合成。
* **特色功能**：結合中心差分法預計算梯度，實現具備動態 Phong Shading 效果的高品質半透明視覺化。

### 3. Streamline
* **技術核心**：RK2 數值積分 (Heun's Method)、幾何著色器 (Geometry Shader)。
* **開發重點**：追蹤 2D 向量場軌跡，實作雙線性內插確保路徑平滑。
* **特色功能**：利用幾何著色器將線段擴展為帶狀矩形，並將流速映射為顏色與線條粗細。

### 4. High Dimensional Data Visualization
* **技術核心**：張量視覺化 (Geometry Shader)。
* **開發重點**：整合資料科學方法與圖形管線，將向量場特徵轉化為橢圓幾何形狀。
* **特色功能**：藉由橢圓的長短軸比例與旋轉角度，視覺化呈現流場的各向異性與應力分佈。

### 5. SOM Surface Fitter
* **技術核心**：自組織映射演算法 (SOM) 、競爭學習機制 、流形網格拓撲 。
* **開發重點**：將無序 3D 點雲數據訓練映射至 $32 \times 32$ 網格 ，並結合動態法向量計算與 Phong Lighting 著色 。
* **特色功能**：支援 Cylindrical (圓柱) 與 Plane (平面) 擬合模式 ，具備即時訓練預覽 與 6 自由度互動相機控制 。
